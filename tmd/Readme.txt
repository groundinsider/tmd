Length.html 파일을 엽니다

왼쪽 메뉴에서 Create, Save, Load, Schedule, Option, Login 중에 하나를 선택할 수 있습니다

[현재 완성되지 않은 것]
1. 완성된 경로 비용 계산 (따라서 Schedule에서 경로 1, 2, 3을 그냥 10, 20, 30으로 설정)
2. 블록 배치 상태 불러오기
3. 시작 지점 블록과 도착 지점 블록은 삭제가 안됨
4. 로그인 한 후에 일정이 추가가 안됨

1. Create

격자 레이아웃에 블록을 놓거나 삭제하려면 먼저 Create 메뉴를 누르고 블록 밑에 데이터를 설정하고 아이콘을 눌러야 합니다

앞에 빨간색, 초록색, 파란색 블록은 "이동 시간 블록"입니다
밑에 분 단위로 데이터를 설정할 수 있습니다

뒤에 노란색 블록 2개는 "시작 지점 블록"과 "도착 지점 블록"입니다
밑에 데이터에서 1부터 3 중 하나를 골라 각자 3개씩만 블록을 놓을 수 있습니다

예를 들어, 시작 지점 블록 밑에 데이터를 1로 설정하고 아이콘을 클릭해서 격자 레이아웃을 누르면 S1이 놓여집니다 
그리고 앞에서 말한 이동 시간 블록 3가지를 이어서 배치해가며 계속 연결하고 끝에 도착 지점 블록 F1을 배치하면 하나의 긴 그래프 모양의 경로가 완성됩니다
S1과 F1으로 만들어진 경로는 경로 1이 됩니다
그 사이에 데이터가 3인 이동 시간 블록이 3개 배치되어 있다면 해당 경로의 총 이동 시간은 9분이 됩니다

이런식으로 경로를 최대 3개까지 만들 수 있습니다

이동 시간 블록인 경우 한 블록 주변에 3개의 블록이 있을 수는 없습니다
따라서 하나의 경로로만 완성해야 합니다

삭제 블록을 선택하고 이미 배치된 블록을 누르면 해당 블록이 삭제됩니다


2. 저장 또는 불러오기

로그인을 한 상태에서 Save 또는 Load 버튼을 누르면 배치한 블록들의 상태가 해당 사용자 계정으로 저장되고 나중에 불러올 수 있습니다


3. 일정 추가

아까 만들었던 경로들을 이용해서 일정을 만들 수 있습니다

메뉴에서 Schedule 버튼을 누르고 일정 내용과 일정 시작 시간, 그리고 아까 미리 만들었던 경로 3가지 중 하나를 고르고 Add를 누르면 출발 시간, 소요 시간, 도착 시간까지 포함된 일정이 추가되고 화면에 나타나게 됩니다 
이런 이동 시간이 포함된 일정은 3개까지만 추가 가능합니다
만약 일정 하나가 끝나는 경우는 다시 추가 가능합니다

경로를 고르지 않고 0으로 두고 Add를 눌러서 일정을 추가하면 일정 시작 시간만 있는 일반 일정이 추가되고 화면에 나타나게 됩니다

이동 시간이 포함된 일정과 일반 일정 모두 합쳐서 최대 20개까지 추가 가능합니다
하나의 일정이 끝나면 다시 추가 가능합니다

이미 추가된 일정과 겹치는 일정은 추가가 안됩니다

최대 5개의 일정이 화면에 보이게 됩니다


4. 옵션

전체 UI의 색을 바꾸거나 글자 크기를 바꿀 수 있습니다


5. 로그인

단순히 아이디와 패스워드를 입력하고 Login 버튼을 누르는 방식입니다
단, 해당 계정이 없는 경우 입력한 아이디와 패스워드로 새로 계정이 생깁니다
계정이 있는 경우는 로그인이 됩니다

로그인을 한 사용자만 데이터를 저장하고 불러올 수 있습니다
